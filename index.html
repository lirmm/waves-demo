Features

  • Create and manage bioinformatic tools execution over such platform as Galaxy server, DRMAA compliant cluster, Direct script execution, API calls to other services...
  • Easily presents these tools in a nice frontend based on Bootstrap3
  • Follow and manage remote REST API access to your service platform
